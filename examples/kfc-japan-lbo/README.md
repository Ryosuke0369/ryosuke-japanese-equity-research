# KFC Holdings Japan — LBO Analysis

## Overview

Full leveraged buyout model reconstructing The Carlyle Group's take-private acquisition of KFC Holdings Japan (TSE: 9873) from Mitsubishi Corporation, announced May 2024 and closed September 2024.

## Model Structure (8 Sheets)

1. **Cover** — Transaction overview and disclaimer
2. **Transaction Assumptions** — TOB pricing, Sources & Uses, debt terms
3. **Income Statement** — 5-year historical + 5-year projections with margin analysis
4. **Balance Sheet** — Pro forma adjustments (goodwill, debt restructuring) + projections
5. **Cash Flow Statement** — Operating / Investing / Financing with BS-CF consistency check
6. **Transaction Summary** — Key metrics dashboard
7. **Debt Schedule** — TLA (amortizing) / TLB (bullet) with credit metrics
8. **Returns Analysis** — MOIC, IRR, and sensitivity tables (exit multiple × exit year)
